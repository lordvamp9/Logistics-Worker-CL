# LogiCore Worker Portal

A logistics management simulation demo featuring:
- **Interactive Map**: Distributor selection with Leaflet.js.
- **RUT Validation**: Chilean ID formatting and validation (Mod 11).
- **Dashboard**: Real-time stats, route simulation, and manifest table.
- **Localized**: Full Spanish interface.

## Quick Start

1. Open `index.html` in your web browser.
2. Select a Base (e.g., LogiCore Santiago).
3. Fill in the form (Format: 12.345.678-k).
4. Access the Dashboard.

## Technologies
- HTML5 / CSS3 (Grid/Flex)
- JavaScript (ES6+)
- Leaflet.js
- Chart.js
- SweetAlert2
